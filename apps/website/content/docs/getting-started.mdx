# Getting Started with Noti

This guide will help you get started with Noti, from installation to creating your first note.

## Installation

### Desktop Application

1. **Download**: Visit the [download page](/#download) and select your platform
2. **Install**:
   - **Windows**: Run the `.exe` installer
   - **macOS**: Open the `.dmg` file and drag Noti to Applications
   - **Linux**: Use the `.AppImage` or `.deb` package

3. **Launch**: Open Noti from your applications menu

### First Launch

When you first launch Noti, you'll be greeted with a welcome screen. You can:

- Create a new note immediately
- Set up your preferred theme
- Configure Git integration (optional)

## Creating Your First Note

1. Click the **"New Note"** button or use `Cmd/Ctrl + N`
2. Give your note a title
3. Start writing in the markdown editor
4. Your note is automatically saved

## Organizing Notes

### Folders

- Create folders by clicking the folder icon
- Drag and drop notes between folders
- Use nested folders for complex organization

### Tags

Add tags to your notes for easier searching:

```markdown
#project #important #todo
```

## Git Integration

Noti includes built-in Git support:

1. **Initialize Repository**: Go to Settings > Git
2. **Automatic Commits**: Noti automatically commits changes
3. **View History**: Click the history icon to see previous versions
4. **Push/Pull**: Sync with remote repositories

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| New Note | `Cmd/Ctrl + N` |
| Search | `Cmd/Ctrl + P` |
| Save | `Cmd/Ctrl + S` |
| Delete Note | `Cmd/Ctrl + D` |
| Toggle Sidebar | `Cmd/Ctrl + B` |
