# Markdown Features

Noti uses TipTap editor with extensive markdown support for beautiful, formatted notes.

## Basic Formatting

### Text Styling

```markdown
**Bold text**
*Italic text*
~~Strikethrough~~
`Inline code`
```

### Headers

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

## Lists

### Unordered Lists

```markdown
- Item 1
- Item 2
  - Nested item
  - Another nested item
```

### Ordered Lists

```markdown
1. First item
2. Second item
3. Third item
```

### Task Lists

```markdown
- [x] Completed task
- [ ] Incomplete task
- [ ] Another task
```

## Code Blocks

Noti supports syntax highlighting for many programming languages:

````markdown
```javascript
function hello() {
  console.log("Hello, world!");
}
```

```python
def hello():
    print("Hello, world!")
```
````

## Tables

Create tables using markdown syntax:

```markdown
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Data 1   | Data 2   | Data 3   |
| Data 4   | Data 5   | Data 6   |
```

## Links and Images

### Links

```markdown
[Link text](https://example.com)
[Internal link](/path/to/note)
```

### Images

```markdown
![Alt text](image-url.png)
```

## Blockquotes

```markdown
> This is a blockquote
> It can span multiple lines
```

## Horizontal Rules

```markdown
---
***
___
```

## Advanced Features

### Math (LaTeX)

```markdown
Inline math: $E = mc^2$

Block math:
$$
\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$$
```

### Callouts

```markdown
> **Note**
> This is an important note

> **Warning**
> This is a warning message
```

## Tips

- Use `Cmd/Ctrl + K` to insert a link quickly
- Press `Tab` to indent lists
- Type `---` and press Enter to create a horizontal rule
- Use markdown shortcuts for faster typing
